Vehicle Sales Information Management System

차량을 판매하는 매니저들이 판매할 차량관리를 용이하게 할수있는 프로그램입니다.

JAVA+DB연동 간단한 CRUD 프로젝트였고 총3명의 인원이 같이 제작하였습니다.

개발 공부를 하면서 만들어 본 첫 프로젝트라 부족하고 아쉬운 부분이 많았던 프로젝트 였습니다.

Technologies
  • - JAVA
  • - Oracle
  • - Eclipse IDE for Enterprise Java and Web Developers - 2022
Project Objectives
Project Structure
  • 프로그램 구조는 차량,맴버(매니저)DAO DTO controller 로 구성하였습니다.
    • Console(Member)
  • 맴버등록,정보확인,수정,삭제(CRUD)
  • 최초 맴버(매니저)등록 후 로그인시 차량 관리 페이지로 넘어가게 됩니다.
    • Console(Car)
  • 차량정보조회,조건조회,차량정보추가,수정,삭제(CRUD)
  • 차량조회시 DB(Oracle)에 저장되어 있는 차량정보 불러오게 됩니다.
